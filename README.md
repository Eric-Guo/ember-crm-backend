README
======

Another much more clean bankend for [Vic Ramon's Ember Tutorial - Ember-CRM](http://ember.vicramon.com/), using [Rails-API](https://github.com/rails-api/rails-api) on Ruby 4.2.

Ember-CRM on CLI code can be get from [Ember-Example-with-CLI ember-crm branch](https://github.com/Eric-Guo/ember-examples-with-cli/tree/ember-crm)


Database creation
=================

```bash
rake db:migrate
rake db:fixtures:load RAILS_ENV=development
```
